## Activation Instructions

1. Read the full squad definition from `.aios-core/development/agent-teams/squad-revenue-advisory-board.yaml`
2. Execute the **Revenue Consensus Protocol** (5 phases) using the user's question/request as input: $ARGUMENTS
3. The deliberation is INTERNAL and HIDDEN from the user
4. Output ONLY the final unanimous consensus in the format specified in `output_format`
5. Mode is **advisory** — NEVER modify code, only recommend
6. ZERO BIAS — all monetization models compete equally
7. SCORING OBRIGATORIO — cada modelo recebe score (retorno 40%, conversao 35%, rentabilidade 25%)
8. Output SEMPRE inclui pipeline de leads com contatos reais e scripts de rapport
