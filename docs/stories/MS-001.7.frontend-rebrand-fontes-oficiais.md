# Story MS-001.7: Frontend Rebrand — PNCP para "Fontes Oficiais" + Indicadores

**Epic:** MS-001 — Multi-Source Integration
**Status:** Ready
**Story Points:** 3
**Prioridade:** P0 — Critica

## Story

**As a** usuario do Descomplicita,
**I want** que a interface reflita que o sistema busca em multiplas fontes oficiais (nao apenas PNCP),
**so that** eu perceba o valor agregado da busca multi-source e tenha visibilidade de quais fontes foram consultadas.

---

## Acceptance Criteria

### Remocao de Mencoes PNCP

1. Todas as 13 mencoes a "PNCP" removidas do frontend
2. Substituicoes conforme tabela acima
3. Nenhum texto hardcoded menciona "PNCP" em nenhum componente

### Indicadores de Fonte nos Resultados

4. Card de resultado exibe badge: "X fontes consultadas" (usando `sources_used.length`)
5. Tooltip ou expandable mostra lista de fontes usadas com status:
   - Verde: fonte retornou dados com sucesso
   - Amarelo: fonte retornou vazio
   - Vermelho: fonte falhou/timeout
6. Contagem por fonte: "PNCP: 150 | Compras.gov: 45 | Querido Diario: 23" (usando `source_stats`)

### Indicadores no Loading

7. Durante fase "fetching", mostrar "Buscando em N fontes oficiais..." em vez de apenas estados
8. Quando fonte completar, mostrar tick visual

### Excel Atualizado

9. Coluna "Fonte" adicionada ao Excel mostrando de qual fonte veio cada registro
10. Se registro apareceu em multiplas fontes, mostrar fonte primaria + "(+N fontes)"

### Atualizacao de Types

11. `types.ts` atualizado com interfaces para `SourceStats` e campos multi-source no result
12. `BuscaResult` expandido com `sources_used: string[]`, `source_stats: Record<string, SourceStats>`, `dedup_removed: number`

### SEO e Meta

13. `layout.tsx` meta description atualizada
14. Nenhuma referencia a "PNCP" no SEO/meta tags

## Technical Notes

### Componente SourceBadges (novo)

```tsx
function SourceBadges({ sources, stats }: { sources: string[]; stats: Record<string, SourceStats> }) {
  return (
    <div className="flex items-center gap-2">
      <span className="text-sm text-ink-muted">{sources.length} fontes consultadas</span>
      {sources.map(s => (
        <span key={s} className="px-2 py-0.5 text-xs rounded-full bg-surface-2">
          {SOURCE_LABELS[s]}: {stats[s]?.total_fetched || 0}
        </span>
      ))}
    </div>
  );
}

const SOURCE_LABELS: Record<string, string> = {
  pncp: "PNCP",
  comprasgov: "Compras.gov",
  transparencia: "Transparência",
  querido_diario: "Diários Oficiais",
  tce_rj: "TCE-RJ",
};
```

### Impacto no Excel (`excel.py`)

```python
# Nova coluna 12: Fonte
ws.cell(row=1, column=12, value="Fonte")
for i, lic in enumerate(licitacoes, start=2):
    source_label = SOURCE_LABELS.get(lic.source, lic.source)
    extra = f" (+{len(lic.sources)-1})" if len(lic.sources) > 1 else ""
    ws.cell(row=i, column=12, value=f"{source_label}{extra}")
```
