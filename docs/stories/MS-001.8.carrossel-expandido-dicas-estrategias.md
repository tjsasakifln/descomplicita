# Story MS-001.8: Carrossel Expandido — 50+ Dicas, Insights e Estrategias

**Epic:** MS-001 — Multi-Source Integration
**Status:** Ready
**Story Points:** 3
**Prioridade:** P1

---

## Story

**As a** usuario do Descomplicita aguardando resultados de busca,
**I want** ver dicas praticas, insights de mercado e estrategias de licitacao no carrossel durante o processamento,
**so that** eu aprenda a competir melhor em licitacoes enquanto espero, transformando tempo de espera em valor educativo.

---

## Story Context

**Existing System Integration:**
- Independente (sem dependencias tecnicas)
- Technology: React, TypeScript
- Touch points: `frontend/app/components/LoadingProgress.tsx`

**Situacao Atual:**
- Carrossel tem 20 "curiosidades" focadas na Lei 14.133/2021 e dados do PNCP
- Conteudo e informativo mas passivo — nao ensina o usuario a agir
- Todas as dicas tem a mesma categoria visual

**Situacao Desejada:**
- 50+ itens organizados em 4 categorias com identidade visual distinta
- Mix de conteudo: legislacao, estrategia, insights de mercado, dicas praticas
- Categorias visualmente diferenciadas com icones e cores
- Rotacao de 6 segundos (levemente mais lento para leitura)

## Acceptance Criteria

### Conteudo Expandido

1. Minimo 50 itens no carrossel (atualmente 20)
2. Conteudo organizado em 4 categorias:

**Categoria 1: Legislacao e Compliance (12+ itens)**
- Artigos relevantes da Lei 14.133/2021
- Mudancas da nova lei vs Lei 8.666
- Prazos legais importantes
- Requisitos de habilitacao
- Exemplo: "A nova lei permite participacao de consorcio de ME/EPP em licitacoes ate R$ 4,8 milhoes, ampliando oportunidades para pequenas empresas."

**Categoria 2: Estrategias de Licitacao (12+ itens)**
- Como preparar propostas competitivas
- Estrategias de precificacao
- Documentacao necessaria
- Erros comuns a evitar
- Exemplo: "Empresas que monitoram licitacoes diariamente encontram 3x mais oportunidades que as que buscam semanalmente."
- Exemplo: "Preparar a documentacao de habilitacao com antecedencia e mante-la atualizada reduz em 70% o risco de desclassificacao."

**Categoria 3: Insights de Mercado (12+ itens)**
- Dados sobre volume de licitacoes
- Setores com mais oportunidades
- Sazonalidade de compras publicas
- Tendencias de mercado
- Exemplo: "Janeiro e fevereiro concentram 35% das licitacoes de uniformes escolares — planeje-se com antecedencia."
- Exemplo: "Prefeituras com menos de 50 mil habitantes representam 60% das licitacoes de uniformes, mas recebem menos propostas."

**Categoria 4: Dicas Praticas do Descomplicita (12+ itens)**
- Como usar filtros de forma eficiente
- Dicas de busca por setor
- Como interpretar resultados
- Funcionalidades do sistema
- Exemplo: "Use o filtro por regiao para focar em estados onde voce tem capacidade logistica de entrega."
- Exemplo: "Salve suas buscas favoritas para acompanhar novas oportunidades sem refazer a pesquisa."

### Identidade Visual por Categoria

3. Cada categoria tem icone e cor de fundo distintos:
   - Legislacao: icone balanca/livro, fundo azul sutil
   - Estrategia: icone alvo/xadrez, fundo verde sutil
   - Insights: icone grafico/tendencia, fundo roxo sutil
   - Dicas Descomplicita: icone lampada/estrela, fundo laranja sutil

4. Label da categoria exibido no card: "Legislacao", "Estrategia", "Insight de Mercado", "Dica Descomplicita"
5. Header do card muda conforme categoria (em vez de fixo "Voce sabia?"):
   - Legislacao: "Voce sabia?"
   - Estrategia: "Estrategia"
   - Insight: "Insight de Mercado"
   - Dica Descomplicita: "Dica"

### Comportamento

6. Rotacao a cada 6 segundos (atual: 5s) para dar mais tempo de leitura
7. Itens exibidos em ordem aleatoria (shuffle no mount), nao sequencial
8. Distribuicao balanceada entre categorias (nao exibir 3 seguidas da mesma)
9. Transicao suave entre itens (fade in/out)

### Atualizacao de Conteudo PNCP

10. Curiosidades existentes que mencionam "PNCP" reformuladas:
    - "O PNCP centraliza..." -> "As fontes oficiais centralizam..."
    - "...consultar licitacoes no PNCP..." -> "...consultar licitacoes em portais oficiais..."
    - etc. (alinhar com MS-001.7)

### Qualidade do Conteudo

11. Todas as dicas tem fonte citada (lei, estatistica, melhores praticas)
12. Nenhuma dica com informacao desatualizada ou incorreta
13. Linguagem acessivel (sem jargao juridico excessivo)
14. Tom pratico e acionavel (usuario consegue agir com base na dica)

### Shuffle Balanceado

```typescript
function shuffleBalanced(items: Curiosidade[]): Curiosidade[] {
  // Agrupar por categoria
  const groups = groupBy(items, i => i.categoria);
  // Intercalar: pegar 1 de cada categoria em round-robin
  const result: Curiosidade[] = [];
  const iterators = Object.values(groups).map(g => shuffleArray(g));
  let idx = 0;
  while (result.length < items.length) {
    const group = iterators[idx % iterators.length];
    if (group.length > 0) result.push(group.shift()!);
    idx++;
  }
  return result;
}
```
